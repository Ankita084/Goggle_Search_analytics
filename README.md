Project Overview
Google Search Analysis is a data analytics and machine learning project that analyzes Google search trends using Python. The project collects real-time search trend data from Google Trends, processes and visualizes the data, and predicts future search interest using machine learning algorithms.
The project also provides an interactive web dashboard using Streamlit where users can explore search trends, compare keyword popularity, and view predictive analytics.

install:
pip install numpy pandas matplotlib seaborn scikit-learn pytrends streamlit
run python analysis.py
streamlit run app.py for stremlite
